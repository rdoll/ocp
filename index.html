Oblivion Character Planner

The essential Elder Scrolls IV: Oblivion character planning tool.

OCP is a browser-based tool that can be used to evaluate the potential maximums of any new or existing TES IV: Oblivion character. Complete usage instructions will accompany the full release, but here are some key points for this fully functional beta:
  1. The basic usage is to input your new or existing character's information into the Character Details panel, set the desired attribute order in the Leveling Order panel, and then view the generated results in the Leveling Results panel. Then, by tweaking the character data and/or leveling order, you can use the automatically regenerated results to tune your character and formulate a rough leveling plan.
  2. The two most important parts of the Leveling Results are any items with a red background (hover over them for an explanation of the issue) and the Spare Skill Points values. Spare Skill Points can be increased without needing their associated attribute bonus; e.g. gaining the inevitable 1-3 Mercantile skill points on levels where Personality is not being raised is safe so long as Personality's Spare Skill Points per Level is 3 or more.
  3. When New Character is selected, the values of Existing Character are ignored and vice-versa.
  4. OCP does not take into account skills/attributes beyond the default maximum of 100 nor does it account for leveling up by losing major skill points from serving a jail sentence.
  5. For details on why OCP is useful in addressing "The Leveling Problem", the articles at Unofficial Elder Scrolls Pages are highly recommended (especially Leveling, Efficient Leveling, and Character Creation).
  6. FireFox 3.5 is the only officially supported browser in this beta release. All other browsers may or may not work (IE is known not to work).
  7. You must allow JavaScript on this server and from o.aolcdn.com. For speed, the AOL CDN provides a "local" copy of the Dojo Toolkit.
  8. The current colors/styles are minimal, yet (hopefully) informative. Upon release, the colors/styles will be made much sexier.
  9. As an example, you can view the complete new and existing character data (including historical leveling details that is lost upon any recalculation) for the toon I created and leveled using OCP as a guide: Nullis.
